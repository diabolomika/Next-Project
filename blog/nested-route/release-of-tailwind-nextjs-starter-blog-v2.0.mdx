## Om denne bloggen -migrert fra Javascript til Typescript

Akkurat tar jeg en utdanning i IT. Denne blogg har en codebase. Tidligere versionen av dene bloggen hadde bade Javascript og Typescript. Type checking og code completion i IDEs er mye bedre med Typescript. Derfor denne bloggen har ingen JS Code lenger. Hvis du vil har mer informasjon om denne bloggen jeg vil gjerne gi deg lit mer informasjon om denne bloggen lagt med Next.Js

Typescript er ogsaa en perfect match for den nye type-safe markdown processor - Contentlayer.

## Contentlayer
[Contentlayer](https://www.contentlayer.dev/) er en innholds-SDK som validerer og transformerer innholdet ditt til typesikre JSON-data som du enkelt kan importere til applikasjonen din. Det gjør det enkelt å jobbe med lokal markdown eller MDX-filer. Dette erstatter `MDX-bundler` og vår egen arbeidsflyt for markdown-behandling.

Først defineres en innholdskilde, som spesifiserer navnet på dokumenttypen, kilden der den er plassert sammen med frontmatter-feltene og eventuelle ekstra beregnede felt som skal genereres som en del av prosessen.
